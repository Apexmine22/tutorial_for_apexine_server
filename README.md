# Инструкция для администрирования Minecraft сервера

## Подключение к серверу по SSH

Подключение к хостингу:

```bash
ssh minecraft@apexine.fun
```
##### Для получения ключей ssh что бы входить в него без пароля ображайтесь к владельцу проекта https://t.me/FrostyB1te



## Работа с screen
### Просмотр активных screen-сессий
```bash
screen -ls
```
#### Показывает список всех активных screen-сессий.

### Создание новой screen-сессии
```bash
screen -S <название_режима>
```

#### Пример:
```bash
screen -S velocity
```

### Используйте понятные названия режимов:
- velocity
- hub
- grief
- anarchy
##### Это помогает не путаться между серверами.


### Подключение к существующей screen-сессии
```bash
screen -x <название_режима>
```

#### Пример:
```bash
screen -S velocity
```

## Запуск Minecraft сервера
```
Без screen запускать сервер запрещено.
При выходе из SSH сервер выключится.
```

### Переход в папку сервера
```
cd servers/<название_сервера>
```

#### Пример:
```
cd servers/hub
```
### Запуск сервера
```
./start.sh
```

##### Скрипт запуска содержит автоматический рестарт сервера при его выключении.

## Полезные команды screen
### Выйти из screen без остановки сервера
#### Нажмите:
```
CTRL + A
D
```
##### После этого screen продолжит работать в фоне.

# ВАЖНО
## Никогда не используйте:
```
CTRL + C
CTRL + Z
```


# Рекомендуемый порядок запуска сервера
```
ssh minecraft@apexine.fun
screen -S hub
cd servers/hub
./start.sh
```


# Рекомендуемый порядок подключения к уже запущенному серверу
```
ssh minecraft@apexine.fun
screen -x hub
```
